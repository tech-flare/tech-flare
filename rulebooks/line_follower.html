Line Follower Event Rule Book

1. Event Overview

  • Event Name: Line Follower Robot
  • Organized By: Swami Vivekanand College of Engineering
  • Participants must design an autonomous robot that follows a predefined line (black on white or white on black).
  • Objective: Complete the track in the shortest possible time.
  • Registration Fee: 200 Rs
  • Event Dates: 23 and 24 September

2. Team Rules

  • Team size: 1–4 members.
  • Only one robot per team.
  • No change of team members after registration.

3. Robot Specifications

  • Only autonomous robots are allowed (no manual control).
  • Size limit: 25cm × 25cm × 25cm, Weight limit: 1.5kg (±5% tolerance).
  • Power: Max 12V DC (onboard battery only).
  • Robot must use sensors (IR/Optical) to detect the line.
  • Must be student-built, not prebuilt.

4. Track Details

  • Background: White with black line (or vice versa).
  • Line width: (as specified in event arena).
  • Track may have curves, T-sections, sharp turns (45°, 90°), junctions, dead ends, and crossings.

5. Game Rules

  • Calibration time of 2 minutes will be given before the start of the run.
  • Each team gets 2 runs; best timing is considered.
  • First round will be the qualifying round.
  • Second round will be the final round.
  • Going off the line → +5 seconds penalty.
  • Each hand touch → +5 seconds penalty.
  • Touching the robot while moving is not allowed.
  • If bot moves out of the track it will be placed at last check point.
  • Damaging the track or unfair play = disqualification.
  • Judges’ decision will be final.

6. Scoring & Winning

  • Winner = fastest run including penalties.
  • In case of a tie → extra run.

7. Safety Rules

  • No harmful materials, fire, or sharp objects.
  • Only safe batteries (Li-ion, Li-Po, NiMH).

8. Code of Conduct

  • Maintain sportsmanship.
  • Organizers may change rules if needed.
